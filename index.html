HTML - HyperText Markup Language
    - Comprised of:
       ○ Text content
       ○ Markup (formatting)
       ○ References to other documents
       ○ References to other HTML documents

Markup
       Tags Content
           Tag + Content+ closing tag = Element'
       Tag examples:
          □ <b> - bold
          □ <em> - emphasis (italics)
             § Without closing tag, opening tag holds to end of doc

HTML Attributes
§ < Tag Attr = 'Value'> contents </tag>
    □ Anchor < a> EX: < a href = "www.reddit.com"> derp </a>
    □ Image < img> EX: < img src="url" alt = "text">
       - No closing tag - "void tag" has no content

White Space
    □ Break < br> - void take, line break
          - Example of inline tag
    □ Paragraph < p> content </p>
          - Example of html block

Span / Div
    - Span - inline
       - < span class = "foo"> text </span>
    - Div - block
       - < div class = "bar"> text < /div>
